# 기능 명세: 생산 라인

## 대상
- `PRODUCING` 상태의 주문(승인 시 재고 부족으로 생산 큐에 등록된 주문)

## 처리 규칙
- 생산 큐는 FIFO(선입선출)로 처리한다.
- 실 생산량 = `ceil(부족분 / 수율)`
- 총 생산 시간(예상 소요) = `평균 생산시간 * 실 생산량`
- 생산 완료 시:
  - 시료 재고를 `실 생산량 - 주문 수량`만큼 순증감(부족했던 만큼 생산 후 주문 수량은 소진, 나머지는 재고로 편입)
  - 해당 주문 상태를 `PRODUCING → CONFIRMED`로 전환

## 조회
- 생산 대기열(큐)에 남아있는 항목을 FIFO 순서로 표시 (주문번호/시료ID/부족분/주문수량)
- 큐가 비어있으면 안내 메시지 표시, 처리 시도 시 오류 없이 "처리할 항목 없음"으로 응답
